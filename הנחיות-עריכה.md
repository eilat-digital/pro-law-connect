# הנחיות עריכה — אתר שירן שושני 🖊️

מדריך קצר לעריכת האתר בכוחות עצמך, בלי ידע בתכנות.

---

## 🚀 איך מריצים את האתר במחשב

1. פותחים טרמינל (PowerShell) בתיקיית הפרויקט `pro-law-connect`
2. פעם ראשונה בלבד: `npm install`
3. מריצים: `npm run dev`
4. פותחים בדפדפן: `http://localhost:8080`

כל שינוי שתשמרי בקבצים יופיע בדפדפן מיד.

---

## 🎛️ פאנל ניהול תוכן (הדרך הקלה ביותר!)

אחרי שהאתר יעלה ל-Netlify, יהיה לך פאנל ניהול בכתובת: **`https://האתר-שלך/admin/`**

בפאנל אפשר לערוך בלי לגעת בקוד: כותרת ראשית, תיאור, אודות, 20 תחומי העיסוק ופרטי יצירת קשר. כל שמירה בפאנל מתעדכנת באתר אוטומטית תוך כ-2 דקות.

**הפעלה חד-פעמית ב-Netlify (אחרי הפריסה הראשונה):**
1. באתר Netlify: Site configuration → Identity → **Enable Identity**
2. שם: Identity → Services → **Enable Git Gateway**
3. Identity → Invitations → **Invite users** → להזמין את המייל שלך
4. פותחים את מייל ההזמנה, קובעים סיסמה — ומתחברים ב-`/admin/`

## 📞 שינוי פרטי קשר, וואטסאפ ורשתות חברתיות

פרטי קשר (טלפון, נייד, פקס, מייל, כתובת) עורכים **בפאנל הניהול** או בקובץ `src/content/homepage.json`.
קישורי סושיאל ווואטסאפ — בקובץ `src/config/site.ts` (ראו בהמשך).

הכול במקום אחד — הקובץ: **`src/config/site.ts`**

שם משנים:
- טלפון, נייד, פקס, מייל, כתובת
- קישור הוואטסאפ (מבוסס על מספר הנייד)
- **קישורי פייסבוק / אינסטגרם / טיקטוק** ← כרגע יש שם כתובות זמניות, חשוב להחליף לכתובות האמיתיות של הדפים שלך!

שינוי בקובץ הזה מתעדכן אוטומטית בכל האתר (תפריט, פוטר, כפתורים צפים, עמודי המשנה).

## ☁️ שמירה והעלאה לאוויר

- שינויים במחשב הם מקומיים בלבד. כדי לגבות אותם ב-GitHub: `git add -A` ואז `git commit -m "תיאור השינוי"` ואז `git push`

## ⚠️ דברים שנשארו פתוחים

1. **קישורי סושיאל אמיתיים** — להחליף ב-`src/config/site.ts`
2. **המלצות לקוחות אמיתיות** — להחליף ב-`src/pages/Index.tsx`
3. **טופס יצירת קשר** — מחובר ל-Netlify Forms. הוא יתחיל לשלוח בפועל רק אחרי שהאתר יעלה לאוויר ב-Netlify (שם גם מגדירים לאיזה מייל יגיעו הפניות: Site → Forms → Notifications).
